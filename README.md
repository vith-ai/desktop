## Features

- **Conversational AI Interface** - Chat with AI to create, analyze, and modify spreadsheets
- **Real-time Spreadsheet Preview** - View and edit Excel files directly in the app
- **Project-based Workflow** - Organize workbooks in project folders
- **Google Sheets Integration** - Import from and export to Google Sheets
- **Business-Focused** - Optimized for spreadsheet analysis and business operations
